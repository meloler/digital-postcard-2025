# Digital Postcard

Interactive Digital Postcard built with Next.js 14+, Tailwind CSS, and Framer Motion.

## Features

- **Apple-Style Scrollytelling**: Smooth parallax, scaling effects, and premium transitions.
- **Cinematic Intro**: Elegant entry sequence.
- **Mobile First**: Optimized for full-screen mobile experiences.
- **Top Tech**: Next.js App Router, Lenis Scroll, Framer Motion.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
